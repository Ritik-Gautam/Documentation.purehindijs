PureHindiJS Documentation

Introduction

PureHindiJS is a JavaScript library that allows developers to write JavaScript code in Hindi. It maps Hindi keywords to their JavaScript equivalents, making coding in Hindi easy and intuitive. This documentation will guide you through the installation, usage, and details of each supported keyword.

Installation

To install PureHindiJS, use npm:

npm install purehindijs

CDN Installation

Alternatively, you can include PureHindiJS directly in your HTML using a CDN:

<script src="https://cdn.jsdelivr.net/npm/purehindijs@1.0.1/index.min.js"></script>

Usage

Import the runHindiScript function from the package:

const { runHindiScript } = require('purehindijs');

Then, you can use it to run Hindi code:

const hindiCode = `
यह संख्या = 5;
लिखो("संख्या है: " + संख्या);
`;

runHindiScript(hindiCode);

Keywords

यह (const)

Use यह to declare a constant in PureHindiJS.

Use चर to declare a variable in PureHindiJS.

Use ये to declare a block-scoped variable in PureHindiJS.

Use अगर to make conditional statements in PureHindiJS.

अगर (संख्या > 10) {
    लिखो("संख्या 10 से बड़ी है");

वरना (else)

Use वरना to define the alternative block of code in an if-else statement.

    लिखो("संख्या 10 से बड़ी है");
} वरना {
    लिखो("संख्या 10 से छोटी या बराबर है");
}

के_लिए (for)

Use के_लिए to create loops in PureHindiJS.

के_लिए (चर i = 0; i < 5; i++) {
    लिखो(i);

Use जबतक to create a while loop in PureHindiJS.

जबतक (i < 5) {

स्विच (switch)

Use स्विच to handle multiple conditions in PureHindiJS.

स्थिति (case)

Use स्थिति inside a switch statement to define cases in PureHindiJS.

कोशिश (try)

Use कोशिश to start a try block for error handling.

कोशिश {
    // कोड जो त्रुटि उत्पन्न कर सकता है
} पकड़ (त्रुटि) {
    लिखो("त्रुटि:", त्रुटि.message);
}

पकड़ (catch)

Use पकड़ to handle errors thrown in a try block.

कोशिश {
    // कोड जो त्रुटि उत्पन्न कर सकता है
} पकड़ (त्रुटि) {
    लिखो("त्रुटि:", त्रुटि.message);
}

अंततः (finally)

Use अंततः to define a block of code that always executes after try and catch.

कोशिश {
    // कोड जो त्रुटि उत्पन्न कर सकता है
} पकड़ (त्रुटि) {

} अंततः {
    लिखो("यह अंततः ब्लॉक है");
}
